# Task 3: Deep-Dive Analysis & Interactive Dashboarding

## Overview

This project demonstrates **Deep-Dive Analysis** and **Interactive Dashboarding** on student performance data. It goes beyond basic EDA (Exploratory Data Analysis) by providing interactive charts to explore data trends and insights.

---

## 1️⃣ What is Deep-Dive Analysis?

Deep-Dive Analysis means analyzing data more thoroughly than standard EDA. While basic EDA includes simple statistics and graphs, Deep-Dive Analysis focuses on **detailed insights** and **interactive visual exploration**.

### Example Analyses in This Task:
- Which student scored the highest marks?  
- What is the average marks of students?  
- Do older students score more marks?  
- How are marks distributed across students?

---

## 2️⃣ What is an Interactive Dashboard?

A **dashboard** is a single screen that displays multiple charts together for easy analysis.  

An **interactive dashboard** allows you to:
- Hover on charts to see details
- Zoom into specific data points
- Move and explore graphs dynamically

### Charts in this Dashboard:
| Chart Type      | What it Shows                     |
|-----------------|----------------------------------|
| **Bar Chart**   | Comparison of student marks      |
| **Histogram**   | Distribution of marks            |
| **Scatter Plot**| Relationship between age and marks |
